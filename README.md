#Introduction 
Скрипт-анализатор предназначен для поиска запрещенных конструкций в файлах разработки. Для проверки:
Скопировать файл GetIllegals.ps1 в папку, папки и файлы которой необходимо проверить (например, в папку Work, где расположены файлы рабочего слоя).
Нажать правой кнопкой и выполнить скрипт с помощью PowerShell. В таком случае будут выведены только грубые ошибки(Вызов серверных функций в обновлении формы или проверки на возможность выполнения действий)
Также скрипт можно запустить из любого места через PowerShell, и в таком случае можно передать параметры запуска –path(указать папку которую необходимо проверить) и –all(Указать All если необходимо также искать не грубые ошибки. После чего на экране отобразятся файлы, в которых допущены ошибки. Без указания параметра –all, часть ошибок отражаться не будет.
